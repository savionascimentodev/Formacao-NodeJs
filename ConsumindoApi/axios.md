### Axios

Para consumir uma Api, existe várias maneiras, como:

- Ajax puro;
- Ajax/Jquery;
- Fetch/Js;
- Axios;

Axios é uma biblioteca moderna e eficiente.

Nessão seção o front-end realizara uma requisição para a ApiGames já desenvolvida e receber seus dados nele.
